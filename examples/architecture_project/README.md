# Architecture Project Example

This fixture follows the user-level Python modeling repository architecture:

```text
analyses/response_study/
  data/
    raw/
    processed/
  results/
    final/
      figures/
      tables/
```

MPLGallery should draft plots from the CSV files under `data/` and
`results/final/tables/`, and show the SVG under `results/final/figures/` as a
reference artifact.
